# Semantic Mapping

## Ontology Selection

This project uses schema.org for general descriptive concepts and health or lifestyle related semantic concepts where suitable. For measurement units, the separate unit mapping document is used.

## Attribute Mapping Table

| Attribute | Meaning | Suggested semantic concept |
|---|---|---|
| ID | Participant identifier | schema:identifier |
| Age | Participant age | schema:age |
| Gender | Participant gender | schema:gender |
| Country | Country of participant | schema:addressCountry |
| Coffee_Intake | Daily coffee intake | food or nutrition related consumption concept |
| Caffeine_mg | Daily caffeine amount | caffeine intake |
| Sleep_Hours | Sleep duration | sleep duration |
| Sleep_Quality | Reported sleep quality | sleep quality |
| BMI | Body Mass Index | body mass index |
| Heart_Rate | Heart rate | heart rate |
| Stress_Level | Stress level | stress level |
| Physical_Activity_Hours | Physical activity duration | physical activity |
| Health_Issues | Reported health issues | health condition |
| Occupation | Participant occupation | schema:occupation |
| Smoking | Smoking status | smoking status |
| Alcohol_Consumption | Alcohol consumption status | alcohol consumption |
| Productivity_Label | Engineered productivity class | derived experimental target |
